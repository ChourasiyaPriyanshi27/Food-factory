# Food Restaurant Website

A modern, responsive food-themed website built with pure HTML and CSS, featuring a clean design with gradient backgrounds and high-quality food imagery.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Clean, contemporary design with smooth animations and hover effects
- **Gradient Background**: Beautiful gradient from light yellow/orange to pink/red
- **High-Quality Images**: Professional food photography from Unsplash
- **Smooth Animations**: CSS animations for enhanced user experience
- **Fixed Navigation**: Sticky header with hover effects and hamburger menu
- **Semantic HTML**: Proper HTML5 semantic structure for accessibility

## Project Structure

```
food-website/
├── index.html          # Main HTML file
├── styles.css          # CSS stylesheet
└── README.md          # Project documentation
```

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with flexbox, gradients, and animations
- **Responsive Design**: Mobile-first approach with media queries

## Getting Started

1. Clone or download the project files
2. Open `index.html` in your web browser
3. No build process or dependencies required!
